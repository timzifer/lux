# Internal Notes

This directory contains working notes, design drafts, and bug-fix
investigations that inform ongoing development of lux. They are kept in
the repository for transparency, but:

- They are **not** part of the public API contract.
- They may be out of date, incomplete, or contradict the current
  implementation.
- They should not be relied on by downstream users.

For stable, user-facing documentation see the top-level [`docs/`](../)
directory and the main [`README.md`](../../README.md).
